
# Billing-Software
Modern Gujarati Business CRM &amp; Address Book with Firebase, Google Sheets Integration, Multi-Category Contact Management, Reports, Excel/PDF Export, and Admin Dashboard.

A modern Gujarati Business CRM & Address Book Web Application designed for traders, wholesalers, retailers, transporters, contractors, and business professionals.

## Features

### Admin Dashboard

* Secure Admin Login
* Change Password
* Dashboard Analytics
* Contact Statistics

### Contact Management

* Add Contact
* Edit Contact
* Delete Contact
* Search Contacts
* Duplicate Detection

### Village Management

* Add New Village
* Edit Village
* Searchable Dropdown
* Auto Selection

### Multi Category Support

* Retail Customer
* Wholesale Customer
* Relative
* Friend
* Transporter
* Religious Organization
* Purchase Merchant
* Salesman
* Contractor
* Labor
* Broker
* General

## Installation

Clone Repository

```bash
git clone https://github.com/your-username/billing-software.git
```

Install Dependencies

```bash
npm install
```

Start Development Server

```bash
npm run dev
```

Build Project

```bash
npm run build
```

---

## Environment Variables

Create .env file

```env
VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_FIREBASE_PROJECT_ID=
VITE_FIREBASE_STORAGE_BUCKET=
VITE_FIREBASE_MESSAGING_SENDER_ID=
VITE_FIREBASE_APP_ID=

GOOGLE_SHEET_WEBHOOK_URL=
```
